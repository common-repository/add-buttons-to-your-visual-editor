Adds buttons to the visual editor, which give you more functionality when editing posts.

== Description ==

This plugin allows you to choose which buttons you want to add to your editors through the Settings < Reading screen. Then it adds them to intuitive locations in the "kitchen sink" row of buttons for all the editors on your site.

== Installation ==

1. Upload the zip file through your plugins->add new screen OR Unzip the archive and upload the sk-add-visual-editor-buttons directory via FTP to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Navigate to your Settings > Reading page and select the buttons you would like to add to your editors

== Frequently Asked Questions ==

= What is this plugin for? =

It gives more functionality to your visual editor when you are writing posts and pages.
